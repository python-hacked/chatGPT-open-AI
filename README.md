# Django Chatbot

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

A Django chatbot project powered by OpenAI's GPT-3 language model.


## Introduction

Django Chatbot is a web application that utilizes OpenAI's GPT-3 language model to provide conversational responses. It acts as a virtual assistant, responding to user input and providing helpful information.

## Features

- Conversational AI: Engage in natural language conversations with the chatbot.
- Real-time Responses: Receive instant responses based on the user's input.
- Customizable Engine: Utilize the power of OpenAI's GPT-3 model to generate accurate and context-aware responses.

## Installation

1. Clone the repository:`https://github.com/python-hacked/chatGPT-open-AI.git`


2. Install the required dependencies:`pip install -r requirements.txt`


3. Set up your OpenAI API key:

- Rename the `.env.example` file to `.env`.
- Replace the `OPENAI_KEY` value with your actual OpenAI API key.

4. Access the chatbot in your web browser at `http://localhost:8000`.

## Usage

1. Enter your message in the input field.
2. Press Enter or click the Submit button to send the message.
3. The chatbot will generate a response based on the input using OpenAI's GPT-3 model.
4. View the chatbot's response in the chat interface.
